ManaAudioPro - デスクトップ版 リアルタイム音楽＆マルチバンドアナライザー 仕様書1. システム概要「ManaAudioPro Desktop」は、Web Audio API とデスクトップ向けアプリケーション・コンテナ技術（Electron / Tauri 等）を融合した、高性能なスタンドアロン型マルチオーディオ解析ソフトウェアです。ボーカルの単一メロディピッチをサブピクセル精度（1Hz未満）で精密抽出するだけでなく、ドラムのキックやスネア、伴奏楽器（コードやベースライン）の周波数スペクトルを全帯域同時に可視化する「マルチバンド・オーディオ・コプロセッサー」です。デスクトップ版として移行したことで、ブラウザ版にあった画面共有ポリシーによるキャプチャ制限を解消し、システム音声のシームレスなループバックキャプチャを実現します。2. デスクトップ特化型アーキテクチャブラウザのサンドボックス（セキュリティ保護）制限から解放され、以下のネイティブパワーを発揮します。① システムオーディオ・ループバック（内部音声直接取得）OSのオーディオスタックと統合することで、Webブラウザのような「画面共有ポップアップ」を挟むことなく、OS全体の出力（ステレオミキサーや仮想オーディオケーブル経由）をアプリケーションに直接パイプインして解析できます。② ハードウェア・アクセラレーション & Retina描画HTML5 Canvas のピクセル解像度をネイティブピクセル比（window.devicePixelRatio）に強制連動。GPU支援による秒間60フレーム（60fps）の超滑らかな描画性能により、動画再生とピッチ軌跡スクロールが完全に同調します。③ Apple macOS ネイティブ・インスパイアデザインmacOS標準のウインドウコントロール（赤・黄・緑のシグナルボタン）を搭載。磨りガラス効果（backdrop-filter）を全面的に活かしたトランスルーセントパネル、極細のインナー境界線（1px solid rgba(255,255,255,0.08)）を用いたラグジュアリーなダークテーマを提供します。3. 主要音響機能＆パイプライン① マルチ入力・デバイスマネージャーデスクトップで利用可能なオーディオ機器を自動的に列挙（Enumerate）します。[ OS 物理マイク / 仮想ミキサー ] ──> [ アプリ内入力デバイスマネージャー ] ──> [ リアルタイム解析 ]
入力セレクトボックス: VB-Audio Cable、BlackHole（Mac）、ステレオミキサーなどの仮想入力デバイスを選択し、デスクトップ音声をシームレスに hot-swap（動的切り替え）できます。ローカルファイル超高速パス: 動画・音声ファイルをコンテナ内に直接ドラッグ＆ドロップ（Blob URL）して高速キャッシング・解析。② 伴奏透視システム：背景スペクトログラム・レイヤー単一のピッチ検出の真後ろに、曲全体のすべての音（ベース、キーボードのコード、シンセサイザー等）の周波数エネルギーをリアルタイムに投影します。仕組み: $F_0$（基本周波数）の検出に加えて、2048ポイントの高速フーリエ変換（FFT）から得られた周波数スペクトルを、12平均律の対数鍵盤スケールにリアルタイムで精密マッピング。グラフィック: 音の強弱に応じて「淡いブルー（弱）」〜「マゼンタ・ピンク（強）」へ連続変化するグラデーションにより、ボーカル（緑のドット）の後ろで「ベース音がどの鍵盤の高さで鳴っているか」「伴奏コードが何度の和音で構築されているか」を網羅的に透視できます。③ 3バンド・ドラムビート分析器 (3-Band Drum-Beat Dynamic Channel)音声信号をデジタルフィルター（BiquadFilterNode）により3系統にリアルタイムに周波数分割（クロスオーバー）し、各帯域のエネルギー・エンベロープを可視化します。                       ┌──> [ Lowpass 150Hz ] ──> LOW チャンネル (キックドラムの鼓動、サブベース音圧)

ビジュアルレベルメーター: 帯域ごとのダイナミクスレベルを3本の独立した超高速LEDバーで表示。ビート鼓動インジケーター: LOW チャンネルの瞬間的なスパイク（キックドラムのアタックなど）を検出し、その音圧に連動して円形ビジュアル（Appleパープル調）が鼓動するように動的拡大スケール（scale）アニメーションを実行します。④ リアルタイム対数スペクトラムアナライザー (Log-Hz Spectrum)KBA（KB）等の著名なビートボックス解説・分析チャンネル等で利用されているものと同等、あるいはそれ以上の解像度を誇る「プロ仕様のリアルタイム・周波数アナライザー」です。低音域（30Hz）から超高音（20,000Hz）までの周波数エネルギーを、対数（Logarithmic）スケールで描画します。対数スケール補間: 人間の耳の周波数知覚特性に合わせて、キックドラムやベースを司る「低音〜中音域（30Hz〜2,000Hz）」を横幅一杯に広く確保し、超高音はギュッとまとめて表示。これによりドラムやベースの具体的なヘルツ帯を一目で精密判定できます。静的Hzガイドライン: 背景に 50Hz, 100Hz, 200Hz, 500Hz, 1kHz, 2kHz, 5kHz, 10kHz, 20kHz のガイドグリッド線と数値ラベルをスタイリッシュに投影。Mana UI ネオン・スペクトラム: Appleパープル、ブルー、ローズピンクを用いた美しいグラデーション波形として滑らかに（秒間60フレーム）描画。⑤ 歌唱・ビートボックス解析システム[ 音声入力 ] ──> [ 自己相関法 ＋ パラボリック補間 ]

高精度ピッチトラッキング: 修正自己相関法にパラボリック（放物線）補間を適用し、通常のFFTの分解能限界を超えた1Hz単位での超高精度F0追従を実現。声区自動判別:Chest Voice (チェスト): $G_3$以下（低音域）Mixed Voice (ミックス): $G\sharp_3$〜$B_4$（中音域）Head / Falsetto (ヘッド/裏声): $C_5$以上（高音域）リアルタイム・ビブラート検出器: 直近600ミリ秒間のピッチ配列から「揺れの周波数（Hz）」と「揺れの幅（Cents）」を秒間60回スピード評価。音楽的に美しいビブラート（4.0Hz〜8.5Hz、幅±15c〜±110c）の発生を自動検出し表示。実効音域（Vocal Range Profile）: セッション中の最低音と最高音を自動記憶（MIDIノート番号）。発声可能な最大オクターブ＆半音スパンを動的に算出表示（「リセット」ボタンで即座に初期化可能）。4. 技術仕様 (Technical Stack)UI/UXコンポーネント: HTML5 / CSS3 (Tailwind CSS JITによるiOS/macOSネイティブデザインの再現)描画コア: Canvas API (高DPI Retinaディスプレイ完全追従、setTransform ダブルバッファリング風)音響解析パイプライン: Web Audio API (低レイテンシ信号プロセッシング)AudioContextAnalyserNode (fftSize: 2048 / 1024 / 256)BiquadFilterNode (Lowpass, Bandpass, Highpass)MediaStreamAudioSourceNode & MediaElementAudioSourceNode (再利用可能な接続ガード設計)開発プラットフォーム移行オプション:Tauri (推奨): バックエンドをRust、フロントエンドを本HTML5で構成。数MBクラスの極限に軽量で安全なデスクトップソフトになります。Electron: Node.js API をネイティブサイドで叩けるため、仮想ミキサー等との親和性が抜群です。
